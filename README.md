# nuxt-gas-webapp

Nuxt.js with Google Apps Script HTML Service


## Prerequisites

First download and install `clasp`

[google/clasp: ðŸ”— Command Line Apps Script Projects](https://github.com/google/clasp)

## Build Setup

``` bash
# change your project id
$ vi clasp.json # change your scriptId

# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

# build for production and deploy google apps script
$ yarn build
$ clasp push
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
